void指针和NULL指针
===================
```
void指针：通用指针，可以指向任意类型，任何类型的指针都可以赋值给void指针
void指针转换成别的类型需要强制类型转换
```
void指针谨慎使用
---------------
```
NULL指针：空指针
***对NULL指针解引用是非法的
注：  
    不清楚要将指针初始化为什么地址时，请将它初始化NULL
    对指针进行解引用时，先检查该指针是否为NULL，节省s大量的调试时间
```
>>>#define NULL ((*void *)0)

NULL与NUL
==========
NUL为空字符
-----------
```
    NULL用于指针和对象，表示控制，指向一个不被使用的地址
    '\0'表示字符串的结尾
```
